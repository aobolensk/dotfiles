# dotfiles

### Dependencies

- required
  - git
  - Python 3.5+
- optional
  - bash
  - i3wm
  - vim
  - tmux
  - [Visual Studio Code](https://code.visualstudio.com/)

#### Ubuntu:
```bash
$ sudo apt install git python3
$ sudo apt install i3 i3-wm py3status i3lock-fancy xautolock
```

### Deploy dotfiles

```bash
$ python3 deploy.py
```

### Update VSCode extensions list

```bash
$ python3 update.py
```
